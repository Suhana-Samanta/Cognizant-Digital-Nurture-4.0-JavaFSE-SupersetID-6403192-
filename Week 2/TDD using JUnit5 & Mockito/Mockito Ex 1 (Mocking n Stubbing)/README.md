# JUnit Testing Assignment

## Overview
This project contains exercises demonstrating Mocking and stubbing an external API using Mockito.



## Prerequisites
- **Java**: JDK 21.
- **VS Code**: VS code.
- **Build Tool**: Maven.
- **Testing Framework**: JUnit 5 (Jupiter).
- **Mocking Library**: Mockito.
